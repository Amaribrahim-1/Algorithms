### Algorithms Performance Comparison

A practical JavaScript script to demonstrate and compare the execution time and time complexity of basic sorting and searching algorithms. 

**What's included:**
* **Sorting:** Bubble Sort vs. Counting Sort.
* **Searching:** Linear Search vs. Binary Search.
* Generates a mock array of 10,000 random integers to properly measure the real-world performance differences between `O(n²)`, `O(n+k)`, `O(n)`, and `O(log n)` complexities.
